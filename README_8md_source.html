1 # Heterogeneous Client Auto Dungeon Master
2 
3 ## Overview
4 
5 This repository provides the code for an auto DM for Dungeons and Dragons 5e via Alexa skill or web interface.
6 
7 The user can request a monster, NPC, encounter, or plot arc by challenge rating, environment, or both. Monsters come
8 from the Monster Manual, Volo's Guide to Monsters, and Mordenkainen's Tome of Foes. Plot arc ideas and multi-monster
9 challenge rating calculations come from the Dungeon Master's Guide.
10 
11 ## Technology
12 
13 The Auto DM is hosted as a Lambda function on AWS. The Alexa skill is hosted by Amazon. The web interface uses an
14 Amazon API Gateway and the demo web version is hosted on GitHub Pages.
